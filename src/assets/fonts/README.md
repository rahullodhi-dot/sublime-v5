# Fonts Directory

## Required Karla Font Files

Please download the Karla font family from Google Fonts and place the following `.ttf` files in this directory:

### Download Link:
https://fonts.google.com/specimen/Karla

### Required Files:
- `Karla-Light.ttf` (Weight: 300)
- `Karla-Regular.ttf` (Weight: 400)
- `Karla-Medium.ttf` (Weight: 500)
- `Karla-SemiBold.ttf` (Weight: 600)
- `Karla-Bold.ttf` (Weight: 700)

### How to Download:
1. Visit the link above
2. Click "Download Family" button (top right)
3. Extract the ZIP file
4. Copy the required `.ttf` files from the extracted folder to this directory
5. The font files are usually in a subfolder named `static/`

### After Adding Files:
The project will automatically use these local font files instead of loading from Google Fonts CDN.
